# 🚐 TravelTrucks

TravelTrucks is a web application for renting camper vans.  
Users can browse available campers, filter them, view detailed information, and submit booking requests.

👉 https://camper-booking-platform.vercel.app/

## 📦 Repository

👉 https://github.com/SerhiiYemets/camper-booking-platform

---

## ✨ Features

- 🔍 Browse camper catalog
- 🎛 Filter campers by:
  - location
  - body type
  - engine type
  - transmission
- 📄 View detailed camper information
- 🖼 Image gallery with preview switching
- ⭐ User reviews
- 📬 Booking form with API integration
- ➕ Load More pagination (infinite scroll via TanStack Query)
- ⚡ Data fetching with React Query

---

## 🛠 Tech Stack

- **Next.js (App Router)**
- **TypeScript**
- **TanStack Query**
- **CSS Modules**
- **React Icons**
- **Vercel** (deployment)
